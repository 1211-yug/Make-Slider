# 🌍 Image Slider

A modern and responsive "Image Slider" built using "HTML, CSS, and JavaScript". This project displays beautiful destinations with smooth sliding animations and navigation controls.

---

## ✨ Features

- 🌍 Beautiful destination images
- ⬅️ Previous & Next navigation buttons
- ✨ Smooth sliding animation
- 📱 Fully Responsive Design
- 🎨 Modern UI with Glassmorphism Effect
- 🖼️ High Quality Images
- 🔄 Infinite Loop Navigation
- 💻 Beginner Friendly Code

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
